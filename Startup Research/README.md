## Описание проекта
Проект направлен на анализ данных о финансировании и покупке стартапов с целью формирования рекомендаций для инвестора.

## Навыки и инструменты
- python
- pandas
- matplotlib

## Общий вывод
В результате проведённого исследования проанализированы данные о финансировании и приобретении стартапов, выявлены типичные объёмы инвестиций, аномальные сделки и наиболее привлекательные категории. Также рассмотрена связь между количеством раундов финансирования и статусом компании, на основе чего сформированы рекомендации для инвестора.
