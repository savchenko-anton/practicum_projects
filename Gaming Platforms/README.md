## Описание проекта
Аналитический проект по исследованию рынка видеоигр за 2000-2013 годы на основе исторических данных о платформах и оценках пользователей и критиков.

## Навыки и инструменты
- python
- pandas

## Общий вывод
В рамках текущего проекта проведен анализ данных об игровых платформах и видеоиграх с 2000 по 2013. Были выявлены наиболее популярные игровые платформы того времени, категоризированы игры в зависимости от оценок пользователей и критиков и даны соответствующие выводы.
